This is a minimal one-page starter template to sell tees via WhatsApp.
Colors: dark green, white, black. Font: minimal luxury serif.

How to use:
1. Replace 'whatsappNumber' in assets/js/script.js with your WA number (country code, no +).
2. Add product images and edit products.json to add new products.
3. Customize index.html text, hero image, and styles in assets/css/style.css.
4. To add payment later: integrate a payment provider and update order flow. For now orders go to WhatsApp.

Files:
- index.html
- assets/css/style.css
- assets/js/script.js
- products.json
- README.md
